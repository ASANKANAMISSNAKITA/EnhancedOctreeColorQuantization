## Setup

1. Create a virtual environment
```bash
python -m venv .venv
```

2. Activate the virtual environment

- Windows (PowerShell)
```powershell
.venv\Scripts\Activate.ps1
```
- Windows (CMD)
```cmd
.venv\Scripts\activate
```
- macOS / Linux
```bash
source .venv/bin/activate
```

3. Install dependencies
```bash
pip install -r requirements.txt
```
